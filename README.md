# 🏎️ PraRoz — Premium Car Dealership Website

موقع واجهة معرض سيارات فاخرة متكامل ومستجيب (Responsive)، تم بناؤه باستخدام التقنيات الأساسية للويب لضمان خفة الوزن والأداء العالي ونظام فلترة فوري ذكي.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)
* **HTML5:** تم التقسيم بنيوياً باستخدام الأوسمة الدلالية الحيثية (`<header>`, `<main>`, `<footer_>`).
* **CSS3:** بناء نظام تخطيط شبكي مرن (Grid & Flexbox)، وإضافة تأثيرات حركية ناعمة (Transitions & Reveal Animations).
* **Vanilla JavaScript:** للتحكم الكامل بـ DOM وصناعة محرك بحث وتصفية متكامل دون الاعتماد على مكاتب خارجية.

---

## ✨ المميزات الرئيسية للمشروع (Key Features)

1. **Combined Search & Filter Engine:** نظام ذكي مدمج في ملف `script.js` يربط بين حقل البحث النصي الفوري وأزرار الفلترة للأقسام (Sedan, SUV, Sport) في دالة موحدة دون حدوث أي تداخل برمي أو تعليق.
2. **Semantic Structure:** كود HTML نظيف جداً ومقسم هندسياً إلى كتل برمجية رئيسية واضحة لتسهيل صيانتها وتطويرها مستقبلاً.
3. **Optimized Login Card:** تم تقليص حجم بطاقة تسجيل الدخول في الواجهة الرئيسية وضبط توازيها البصري وسحبها لجهة اليمين لتعطي توازناً فنياً رائعاً مع نصوص الترحيب الفاخرة.
4. **Responsive Web Design (RWD):** متوافق بنسبة 100% مع شاشات الهواتف الذكية والأجهزة اللوحية عبر شريط تنقل بري ذكي (Hamburger Menu).
